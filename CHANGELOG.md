# Changelog

All notable changes to Azure Cosmos DB Shell will be documented in this file.

## [0.31.1] - 2026-02-04

### Added
- Initial private preview release
- MCP (Model Context Protocol) server support
- VS Code extension integration
- Bash-like navigation commands (`ls`, `cd`)
- Full CRUD operations for databases, containers, and items
- Query execution with SQL syntax
- Pipeline composition with JSON data flow
- Scripting support with variables, control flow, and custom functions
- Azure AD authentication support

### Security
- Local-only MCP server binding
- Configurable MCP port
- Support for Azure AD/Entra ID authentication
